# Brain Vault Starter

> [!summary] Resumen
> Guía de inicio rápido para montar una bóveda de Obsidian (segundo cerebro) desde cero. Incluye una estructura de carpetas mínima con plantillas para registro diario, ideas, tareas, objetivos y contactos, más instrucciones para conectarla con Claude Code.

Esta es una estructura de carpetas inicial para tu bóveda de Obsidian. Coloca estas carpetas en la bóveda que acabas de crear y tendrás un segundo cerebro funcionando desde el primer día.

No tienes que usarlas todas. Elige las 2 o 3 que coincidan con cómo funciona realmente tu cerebro e ignora el resto. La idea es que esto se adapte a ti, no al revés.

## Estructura de carpetas

```
my-brain/
├── daily-log/
│   └── 2026-05-11.md
├── ideas/
│   └── ideas.md
├── tasks/
│   └── tasks.md
├── goals/
│   └── goals.md
├── notes/
│   └── notes.md
└── people/
    └── people.md
```

## Plantilla de registro diario

Coloca esto en `daily-log/2026-05-11.md` (o la fecha de hoy). Crea un archivo nuevo como este cada día. Incluso una línea es suficiente.

```
# 2026-05-11

## Qué hice hoy
-

## En qué estoy pensando
-

## Ideas que merecen guardarse
-

## Me ronda la cabeza
-
```

## Plantilla de archivo de ideas

Un solo archivo continuo. Simplemente sigue añadiendo al principio.

```
# Ideas

## 2026-05-11
- Idea aquí
- Otra idea aquí

## 2026-05-10
- Idea anterior
```

## Plantilla de archivo de tareas

```
# Tareas

## Esta semana
- [ ] Cosa por hacer
- [ ] Otra cosa

## Algún día
- [ ] Algo más grande que quiero abordar
```

## Cómo conectar esto con Claude Code

Una vez que tu bóveda tenga archivos, abre una sesión nueva de Claude Code en cualquier proyecto y pega esto:

```
Tengo una bóveda de Obsidian en [PEGA AQUÍ LA RUTA DE TU BÓVEDA]. Actualiza mi CLAUDE.md global (~/.claude/CLAUDE.md) para que revises esta bóveda cada vez que haga referencia a mis notas, mi diario, mis ideas, mis tareas o mi cerebro. Enumera las carpetas para que sepas lo que hay dentro.

Confirma lo que has guardado y dime cómo sabré que lo estás comprobando.
```

## Qué hacer mañana

Abre Obsidian. Crea un nuevo archivo de registro diario con la fecha de hoy. Escribe una frase sobre lo que te ronda la cabeza. Eso es todo.

La mayoría de la gente lo deja después de una semana. Los que siguen durante 6 meses tienen una IA que les conoce mejor que nadie en su vida. Incluido su terapeuta.

Este es el archivo. El resto es solo presentarte.
